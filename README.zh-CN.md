# 交易所账户审核与申诉材料准备工具包

**适用于币安、欧易 OKX、Bybit、Coinbase、Kraken、Gate、KuCoin、Bitget 等平台账户风控、冻结、限制、提现审核、资金来源审查、C2C/P2P 限制等场景的材料准备。**

本仓库由 **Delta Capital HK / 德尔泰 Delta&Capital** 发布，目标是帮助用户在遇到交易所账户限制、风控审核、提现延迟、资金来源审查或补充材料要求时，把事实、时间线、资金来源、交易路径和证据整理成清晰的申诉材料包。

本工具包不是交易所官方流程，不用于绕过风控，不承诺解冻、放款、提现通过或追回成功。它的作用是帮助用户更清楚地准备材料、解释事实、减少无效沟通。

## 先做什么

遇到账户限制、提现审核或资金冻结时，不建议只反复催客服。先整理以下信息：

1. 平台名称、账户 UID、注册邮箱或手机号
2. 被限制的功能：登录、交易、提现、充值、P2P/C2C、API 等
3. 平台提示原文截图、工单编号、申诉编号
4. 限制发生时间和完整时间线
5. 涉及订单号、交易哈希、钱包地址、币种和金额
6. 资金来源说明和证明材料
7. 与平台客服、交易对手、付款方的沟通记录
8. 身份认证、收入证明、银行流水、合同、发票等补充材料
