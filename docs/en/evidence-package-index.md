# Evidence Package Index

Use this index to organize documents before submitting an appeal, replying to support, or sharing materials with a professional advisor.

## A. Account and Platform Information

| ID | Material | Notes |
|---|---|---|
| A01 | Account UID screenshot | Show account identifier |
| A02 | Restriction notice screenshot | Exact platform message |
| A03 | Support ticket or appeal ID | Include timestamp |
| A04 | Affected function screenshot | Withdrawal, P2P, trading, etc. |

## B. Identity and Account Ownership

| ID | Material | Notes |
|---|---|---|
| B01 | KYC status screenshot | If available |
| B02 | Registered email or phone proof | Avoid unnecessary private exposure |
| B03 | Account profile screenshot | Redact unrelated sensitive data |

## C. Source of Funds

| ID | Material | Notes |
|---|---|---|
| C01 | Source of funds explanation | Use template |
| C02 | Bank statement or income proof | Relevant period only |
| C03 | Contract, invoice, or payment note | If business-related |
| C04 | OTC/P2P order record | Include order ID |
| C05 | Exchange order history | Spot/futures/convert history if relevant |

## D. On-chain Transaction Evidence

| ID | Material | Notes |
|---|---|---|
| D01 | Transaction hash list | Include chain name |
| D02 | Wallet address explanation | Ownership or relationship |
| D03 | Blockchain explorer screenshots | Include timestamp and URL |
| D04 | Fund flow notes | Simple flow description |

## E. Communication Records

| ID | Material | Notes |
|---|---|---|
| E01 | Platform support emails | Include full thread |
| E02 | App support chat screenshots | Include timestamps |
| E03 | Counterparty communication | If relevant |
| E04 | Payment confirmation | If fiat/P2P-related |

## F. Submission Log

| Date | Submitted To | Materials Submitted | Result / Next Step |
|---|---|---|---|
| YYYY-MM-DD | Platform support | A01, A02, C01 | Awaiting review |
