# 香港企业跨国协查账户限制解除案例

本页面根据真实项目材料脱敏整理，适用于企业交易所账户因跨国协查、平台法务复核、资金来源审查或提现限制而长期无法转出的场景。

## 脱敏案例摘要

一家香港注册企业的交易所账户内显示约 103 万 USDT，资产可见但提现受限。该案不只是普通账户风控，还涉及跨国协查、企业主体材料、链上资金来源、平台法务复核和出金安全安排。

在德尔泰（Delta&Capital）介入前，企业已通过其他专业渠道与平台沟通，但材料未能形成平台可复核的完整事实链。公开材料层面，该类案件需要把企业身份、账户控制关系、资金来源、交易背景、链上记录和平台回复整理成统一材料包。

公开版案例链路可概括为：

```text
香港企业账户
  -> 交易所资产可见但提现受限
  -> 平台法务和跨国协查因素叠加
  -> 企业主体和资金来源需要重新整理
  -> 平台复核后恢复资金处理路径
```

## 公开难点

- 企业账户比个人账户更重视主体身份、授权关系和资金来源。
- 平台通常需要看见企业材料、交易背景和链上记录之间的对应关系。
- 单独提交律师函或客服沟通记录，不一定能解释完整事实链。
- 跨国协查场景下，不同机构看到的材料口径必须一致。

## 可公开整理的材料方向

| 材料 | 说明 |
|---|---|
| 企业主体材料 | 注册资料、授权代表、账户控制关系 |
| 账户和限制说明 | UID、资产状态、受限功能、平台回复 |
| 资金来源材料 | 企业经营、交易、收付款和资产形成说明 |
| 链上交易记录 | 充值、提现、钱包地址、交易哈希清单 |
| 沟通记录 | 平台法务、客服、律师或其他机构往来摘要 |
| 证据目录 | 用统一编号组织全部附件和说明 |

## 常见错误

- 只提交企业证件，没有解释账户资金来源。
- 只提交链上记录，没有说明企业交易背景。
- 不同渠道材料口径不一致。
- 没有把平台问题拆成“身份、资金、交易、风险、请求事项”几个部分。

## 公开边界

本页面不披露企业名称、交易所名称、UID、真实地址、交易哈希、法律文书、协查细节或内部材料组织方法。

## 重要声明

企业账户限制的处理结果由交易所、平台法务、执法机关或其他第三方决定。本案例仅说明公开材料整理方向，不构成解冻承诺。
