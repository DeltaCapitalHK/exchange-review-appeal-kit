# 欧洲外部司法调查冻结解除案例

本页面根据真实项目材料脱敏整理，适用于交易所账户因外部司法机关调查而被冻结、账户功能受限、平台提示 external agency 或 law enforcement request 的场景。

## 脱敏案例摘要

某用户交易所账户内约 960 万 USDT 资产被限制，账户功能受限。与普通平台风控不同，该案的限制来源并非交易所算法审核，而是欧洲司法机关外部调查。用户初期并不知道账户为何被冻结，只能看到平台侧功能受限。

德尔泰（Delta&Capital）介入后，首先将案件从“平台风控申诉”区分为“外部司法调查冻结”场景。公开材料层面，需要围绕账户基本事实、资金来源、交易背景、平台限制原因、司法协查状态和可提交证据建立清晰目录。后续处理依赖司法机关、平台法务、当地程序和正式文书，不是普通客服催办可以解决。

公开版案例链路可概括为：

```text
交易所账户资产受限
  -> 平台提示外部机构调查
  -> 确认不是普通平台风控
  -> 整理资金来源和交易背景材料
  -> 等待司法和平台程序闭环
```

## 公开难点

- 用户容易把外部司法冻结误认为普通平台风控。
- 交易所通常不能单方面解除司法机关要求的限制。
- 材料必须区分事实、解释、证明和待核实事项。
- 跨语言、跨法域、跨机构沟通对文书和证据一致性要求较高。

## 可公开整理的材料方向

| 材料 | 说明 |
|---|---|
| 平台限制提示 | 平台显示的限制原因、工单或邮件摘要 |
| 账户基础事实 | UID、账户主体、资产余额、受限功能 |
| 资金来源说明 | 入金来源、交易背景、资产形成过程 |
| 交易记录清单 | 平台订单、充值提现、链上记录 |
| 身份和主体材料 | 个人或企业身份、地址、授权材料 |
| 程序进展记录 | 平台、律师、司法机关或其他机构反馈 |

## 常见错误

- 持续向一线客服要求“马上解冻”，但没有处理外部司法来源。
- 没有保存平台关于 external agency / law enforcement 的提示。
- 提交零散截图，缺少资金来源和交易背景说明。
- 在未确认情况下把司法冻结写成平台误伤。

## 公开边界

本页面不披露具体司法文书、案号、交易所名称、客户身份、真实资金路径、律师沟通内容或内部协作方法。

## 重要声明

外部司法调查冻结的解除通常取决于司法机关、平台法务和相关程序结果。德尔泰可协助整理材料和流程支持，但不承诺任何司法或平台处理结果。
