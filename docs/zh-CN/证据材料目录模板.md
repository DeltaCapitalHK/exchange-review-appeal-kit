# 证据材料目录模板

在提交交易所申诉、回复客服、咨询专业顾问前，可以先按以下方式整理证据材料。

## A. 账户与平台信息

| 编号 | 材料 | 备注 |
|---|---|---|
| A01 | 账户 UID 截图 | 显示账户标识 |
| A02 | 平台限制提示截图 | 保留提示原文 |
| A03 | 工单或申诉编号 | 包含提交时间 |
| A04 | 被限制功能截图 | 提现、P2P、交易等 |

## B. 身份与账户归属

| 编号 | 材料 | 备注 |
|---|---|---|
| B01 | KYC 状态截图 | 如可提供 |
| B02 | 注册邮箱或手机号归属证明 | 避免暴露无关隐私 |
| B03 | 账户资料页面截图 | 可遮盖无关敏感信息 |

## C. 资金来源

| 编号 | 材料 | 备注 |
|---|---|---|
| C01 | 资金来源说明 | 使用模板 |
| C02 | 银行流水或收入证明 | 只提供相关时间段 |
| C03 | 合同、发票或付款说明 | 经营收入场景 |
| C04 | OTC/P2P 订单记录 | 包含订单号 |
| C05 | 交易所订单历史 | 如现货、合约、兑换记录 |

## D. 链上交易证据

| 编号 | 材料 | 备注 |
|---|---|---|
| D01 | 交易哈希列表 | 标明链名称 |
| D02 | 钱包地址说明 | 说明地址归属或关系 |
| D03 | 区块浏览器截图 | 包含 URL 和时间 |
| D04 | 资金流向说明 | 用简单文字或图示 |

## E. 沟通记录

| 编号 | 材料 | 备注 |
|---|---|---|
| E01 | 平台客服邮件 | 保留完整线程 |
| E02 | App 在线客服截图 | 包含时间 |
| E03 | 与交易对手沟通记录 | 如 P2P/OTC 相关 |
| E04 | 付款凭证 | 如法币或 P2P 相关 |

## F. 提交记录

| 日期 | 提交对象 | 提交材料 | 结果 / 下一步 |
|---|---|---|---|
| 2026-xx-xx | 平台客服 | A01, A02, C01 | 等待审核 |

## 整理建议

文件名尽量清楚，例如：

```text
A01-账户UID截图.png
A02-平台限制提示截图.png
C01-资金来源说明.pdf
D01-交易哈希列表.xlsx
E01-客服邮件记录.pdf
```

不要把所有截图打包成一个无序压缩包。审核人员越容易理解材料，沟通效率通常越高。
