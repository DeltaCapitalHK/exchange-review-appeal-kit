# 资金来源说明模板

## 目的

当交易所要求解释资金来源、交易目的、钱包归属或相关链上活动时，可以使用本模板。

## 适用场景

当交易所、钱包服务商或交易平台要求说明充值、提现、OTC/P2P 交易、钱包转账或账户活动时，可以使用本模板。

## 常见错误

- 只说“资金合法”但不提供证明。
- 把无关交易混入同一份说明。
- 提交截图但没有订单号、交易哈希或日期。
- 多次提交互相矛盾的说法。

## 3. 交易路径说明

1. 资金来源于：__________
2. 资金于 __________ 进入 __________ 钱包/平台账户；
3. 资金通过交易哈希/订单号 __________ 转出或转入；
4. 资金于 __________ 进入当前平台/账户；
5. 资金用途为：__________

## 4. 附件材料

本说明附带以下材料：

- 银行流水或收入证明
- 合同、发票或付款说明
- 交易所订单截图
- OTC/P2P 订单截图
- 钱包地址说明
- 区块浏览器截图
- 与交易对手的沟通记录
- 其他相关证明
