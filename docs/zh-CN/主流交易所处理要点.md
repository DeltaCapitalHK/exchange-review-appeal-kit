# 主流交易所处理要点

本页整理币安、欧易 OKX、Bybit、Coinbase、Kraken、Gate、KuCoin、Bitget 等平台账户风控、冻结、提现审核、资金来源审查时常见的材料准备思路。

本页不是交易所官方说明，也不替代平台客服。最终以账户内提示和官方帮助中心为准。

## 币安 Binance 被风控 / 提现审核 / 账户限制

常见情况可能包括：提现审核、账户限制、P2P 功能限制、补充 KYC、资金来源审查、风险控制检查等。

建议准备：

- Binance UID
- 被影响功能：提现、现货、合约、P2P、登录、API、法币等
- 平台提示原文截图
- 是否出现 Appeal / 申诉入口
- 相关订单号
- 充值、提现记录
- TxID / 链上交易哈希
- 资金来源说明
- 平台要求的身份、付款、订单或资金证明截图

官方帮助中心：https://www.binance.com/en/support

## 欧易 OKX 被风控 / C2C 资金锁定 / 提现限制

常见情况可能包括：账户审核、提现审核、C2C/P2P 限制、资产锁定、风控检查、补充验证等。

建议准备：

- OKX UID
- 被影响功能：提现、C2C/P2P、交易、登录、API、法币等
- 账户内显示的锁定或限制类型
- C2C 订单号或资产记录
- 付款凭证
- 与交易对手沟通记录
- 资金来源说明
- 限制提示和客服回复截图

官方帮助中心：https://www.okx.com/help

## Bybit 账户限制 / 提现审核 / P2P 争议

常见情况可能包括：提现审核、P2P 订单争议、账户风险审核、补充验证、功能限制等。

建议准备：

- Bybit UID
- 被限制功能和提示原文
- 订单号、P2P 订单号或交易记录
- 充值、提现记录
- TxID / 交易哈希
- P2P 付款凭证和对手方沟通记录
- 资金来源说明
- 限制发生前后的操作时间线

官方帮助中心：https://www.bybit.com/en/help-center/

## Coinbase 账户 under review / 限制发送或提现

常见情况可能包括：账户审核、发送/接收限制、身份验证审核、资金来源审查、合规相关 hold 等。

建议准备：

- Coinbase 注册邮箱
- 被影响功能：买入、卖出、发送、接收、提现、登录等
- App 或网页内显示的提示原文
- 限制发生时间范围
- 交易 ID 或链上交易哈希
- 身份验证状态
- 资金来源说明
- 官方通知截图

官方帮助中心：https://help.coinbase.com/

## Kraken 账户冻结 / 提现 hold / 资金审核

常见情况可能包括：账户验证审核、提现 hold、入金审核、资金来源要求、合规检查等。

建议准备：

- Kraken public account ID 或工单编号
- 被影响功能
- 平台提示或客服要求原文
- 入金记录
- 交易和提现历史
- 资金来源说明
- Kraken 要求的补充材料

官方支持：https://support.kraken.com/

## Gate、KuCoin、Bitget、MEXC、HTX 等其他平台

其他平台也可以按同样逻辑准备：

- 账户 UID
- 被影响功能
- 限制提示原文
- 工单或申诉编号
- 订单和交易记录
- 链上交易哈希
- 资金来源说明
- 付款和交易对手证明
- 平台通知截图

官方参考：

- Gate Help Center: https://www.gate.com/help
- KuCoin Help Center: https://www.kucoin.com/support
- Bitget Support: https://www.bitget.com/support

## 重要提醒

不要相信任何声称可以“绕过交易所风控”“保证解冻”“内部通道放款”的说法。正规处理路径通常需要事实说明、身份/账户核验、资金来源材料、交易记录和平台审核。
