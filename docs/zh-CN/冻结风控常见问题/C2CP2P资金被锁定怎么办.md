# C2C/P2P 资金被锁定怎么办

C2C/P2P 资金被锁定通常与订单争议、付款来源、交易对手投诉、银行风控、平台合规审查或异常交易有关。处理时应把订单、付款、聊天和平台沟通记录完整整理出来。

## 先确认锁定原因

- 是买币订单、卖币订单，还是收款后账户被限制。
- 对手方是否发起申诉、撤回付款或声称未收到款。
- 银行卡、支付账户或交易所账户是否同时被限制。
- 平台要求补充的是付款证明、收款证明、身份信息还是交易说明。

## 材料清单

- 平台名称、UID、订单号。
- C2C/P2P 订单详情截图。
- 付款或收款凭证。
- 银行流水、支付账户记录。
- 与交易对手的聊天记录。
- 平台客服、工单、申诉记录。
- 资金来源和交易目的说明。
- 如涉及警方或银行通知，保存相关文书或编号。

## 说明建议

```text
订单号:
交易方向:
交易金额:
付款/收款方式:
对手方信息:
争议原因:
已提供凭证:
希望平台复核事项:
```

## 常见错误

- 删除与对手方的聊天记录。
- 只提供付款截图，不提供订单号。
- 不说明银行卡或支付账户是否被银行限制。
- 在平台外私下和解但不保留凭证。
