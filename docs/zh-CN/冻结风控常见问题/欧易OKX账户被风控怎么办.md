# 欧易 OKX 账户被风控怎么办

OKX 账户被风控、提现延迟、C2C 资金锁定或要求补充材料时，应先根据账户内提示和官方客服要求准备材料。公开资料只能提供材料整理方向，不能保证解除风控。

## 需要先看清楚

- 限制的是提现、交易、充值、C2C/P2P，还是账户整体。
- 平台是否要求身份验证、资金来源、订单说明或链上交易解释。
- 是否涉及某一笔充值、提现、C2C 订单或链上地址。
- 是否已有工单号、申诉入口、邮件或客服回复。

## 建议准备材料

- OKX UID、注册邮箱或手机号。
- 风控提示截图、邮件或客服原文。
- 涉及订单号、充值记录、提现记录。
- 链上交易哈希、钱包地址、币种、金额、时间。
- 法币入金、买币、卖币、银行流水或其他资金来源证明。
- C2C/P2P 对手方沟通记录，如涉及。
- 对每笔异常或被询问交易的简短解释。

## 说明结构

```text
1. 账户和限制情况
2. 涉及交易或订单
3. 资金来源说明
4. 交易目的说明
5. 已提交或可补充的证明材料
6. 希望平台复核的事项
```

## 常见错误

- 只催客服，不补材料。
- 对同一笔资金来源给出多个版本。
- 忽略 C2C/P2P 对手方付款或收款记录。
- 只提供截图，没有订单号或交易哈希。
