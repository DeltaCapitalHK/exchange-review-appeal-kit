## 先准备什么

1. 平台名称、账户 UID、注册邮箱或手机号。
2. 被限制的功能，例如提现、交易、登录、P2P/C2C、API。
3. 平台提示原文、邮件、工单号、申诉编号。
4. 限制发生时间、涉及订单、交易哈希、钱包地址和金额。
5. 资金来源说明、交易目的说明、相关证明材料。
6. 与平台客服、交易对手、付款方或收款方的沟通记录。
