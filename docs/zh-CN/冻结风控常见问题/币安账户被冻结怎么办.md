# 币安账户被冻结怎么办

币安账户被冻结、提现被限制或进入风险审核时，先确认平台提示原因和要求补充的材料。不要反复提交情绪化信息，应把账户、交易、资金来源和申诉请求整理成清晰材料包。

## 先确认冻结类型

- 是否无法登录账户。
- 是否可以登录但不能提现。
- 是否 P2P/C2C 功能被限制。
- 是否某一笔充值、提现或交易被要求说明。
- 是否要求补充 KYC、地址证明、资金来源或交易目的。
- 是否已有工单号、邮件编号或申诉入口。

## 材料清单

| 材料 | 说明 |
|---|---|
| 账户信息 | UID、注册邮箱或手机号、工单号 |
| 平台提示 | 冻结、限制、审核页面截图或邮件原文 |
| 时间线 | 何时发现限制、何时提交过材料 |
| 相关交易 | 订单号、交易哈希、充值/提现记录 |
| 资金来源 | 买币记录、银行流水、收入证明、交易所提现记录等 |
| 交易目的 | 为什么充值、提现、换币或转账 |
| 沟通记录 | 客服回复、邮件、P2P 对手方沟通 |

## 申诉说明建议

```text
我希望申请复核账户限制。以下为账户信息、限制情况、相关交易、资金来源说明和可提供证明材料。

UID:
限制功能:
发现时间:
相关订单/交易:
资金来源:
交易目的:
已附材料:
希望平台复核事项:
```

## 常见错误

- 只写“请解冻”，不说明资金来源和交易背景。
- 多次提交互相矛盾的解释。
- 用大量截图代替交易哈希和订单号。
- 在非官方渠道提交密码、验证码或完整敏感信息。

## 德尔泰可以协助什么

德尔泰（Delta&Capital）可以协助整理币安账户冻结相关的材料目录、资金来源说明、交易解释和申诉文本。最终是否解除限制，由平台根据其审核规则决定。
