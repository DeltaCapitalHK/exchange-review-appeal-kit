# Bybit 提现审核一直不通过怎么办

Bybit 或其他交易所提现审核延迟时，先确认是否为普通安全审核、账户风控、资金来源审查、地址风险审查或补充 KYC。不同情况需要的材料不同。

## 先确认

- 提现是否显示 pending、under review、suspended 或 rejected。
- 是否只有某个币种、某个地址或某笔金额被限制。
- 平台是否要求补充身份、资金来源、地址用途或交易目的。
- 是否有工单号、邮件、客服聊天记录。

## 需要准备

| 材料 | 说明 |
|---|---|
| 账户信息 | UID、注册邮箱、工单号 |
| 提现记录 | 币种、金额、网络、地址、申请时间 |
| 资金来源 | 入金、交易、兑换、转账来源说明 |
| 地址用途 | 提现到个人钱包、其他交易所或业务地址的原因 |
| 链上记录 | 如涉及充值来源，提供哈希和地址 |
| 证明材料 | 银行流水、买币记录、收入证明、合同等 |

## 写给平台时避免

- 不要写“马上放款，否则投诉”这种无材料表达。
- 不要提交无法解释的第三方收付款记录。
- 不要隐藏与审核有关的充值或提现。
- 不要反复更换提现地址制造更多风险信号。
