# 交易所要求资金来源说明怎么写

交易所要求资金来源说明时，重点是解释资金从哪里来、为什么进入交易所、经过哪些账户或钱包、是否能提供证明。说明应真实、清晰、前后一致。

## 建议结构

```text
1. 账户基本信息
2. 资金来源类型
3. 入金或充值路径
4. 相关交易记录
5. 资金用途或交易目的
6. 可提供证明材料
7. 请求平台复核事项
```

## 常见资金来源类型

- 工资、经营收入、服务收入。
- 投资赎回、历史持仓、交易收益。
- 交易所之间转账。
- 自有钱包转入。
- P2P/C2C 买币或卖币。
- 借款、赠与、公司资金或其他来源。

## 材料示例

| 来源类型 | 可准备材料 |
|---|---|
| 工资收入 | 工资流水、雇佣证明、纳税记录 |
| 经营收入 | 合同、发票、公司流水、收款记录 |
| 交易收益 | 交易记录、成交明细、历史持仓 |
| 交易所转入 | 提现记录、充值记录、账户 UID |
| P2P/C2C | 订单号、付款凭证、聊天记录 |
| 链上钱包 | 钱包地址、交易哈希、资金流向说明 |

## 常见错误

- 只写“自有资金”，没有任何证明。
- 资金来源和交易记录时间对不上。
- 忽略中间账户、钱包或交易所路径。
- 提交不真实或无法解释的材料。
