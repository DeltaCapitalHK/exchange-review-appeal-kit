# 交易所申诉邮件怎么写

交易所申诉邮件或工单回复应简洁、真实、结构清楚。不要只表达焦虑或投诉，应把平台需要审核的事实和材料列出来。

## 推荐结构

```text
Subject: Request for account review / withdrawal review

1. Account information
2. Restriction or review status
3. Relevant transaction or order records
4. Source of funds explanation
5. Purpose of transaction
6. Attached evidence list
7. Requested review action
```

## 中文模板

```text
我的账户目前出现 [提现审核/账户限制/风控审核/C2C 资金锁定]。以下为我整理的说明和证明材料，请协助复核。

账户信息：
- 平台：
- UID：
- 注册邮箱/手机号：
- 工单号：

限制情况：
- 发现时间：
- 受影响功能：
- 平台提示内容：

相关交易：
- 日期：
- 币种：
- 金额：
- 网络：
- 交易哈希/订单号：

资金来源说明：
[简要说明资金来源，并列出证明材料]

交易目的说明：
[说明充值、提现、换币或转账目的]

附件目录：
1.
2.
3.

希望平台协助复核账户限制/提现审核，并告知是否还需要补充其他材料。

谢谢。
```

## 常见错误

- 邮件太长但没有关键字段。
- 只表达“很着急”，没有证据。
- 附件无命名，客服无法对应材料。
- 每次回复都换一种说法。
